# 📚 StudyMate AI

An AI-powered study platform that helps students upload PDFs and learn smarter using AI-generated summaries, quizzes, flashcards, and interactive chat.

## 🚀 Planned Features

- User Authentication
- Upload PDF Notes
- AI Chat with PDFs
- AI Notes Summarizer
- AI Quiz Generator
- AI Flashcards
- Study Planner
- Study History
- Responsive Dashboard

## 🛠️ Tech Stack

### Frontend
- React
- Tailwind CSS

### Backend
- Node.js
- Express.js

### Database
- MongoDB

### Authentication
- JWT

### AI
- LLM API
